  • InputField
    • Inserts a form input field.

    Parameters

    name - the field name value - optional value

    Usage:

    [{InputField name="myField"}]

    renderer.sjsname = extension.getAttribute("name"); value = extension.getAttribute("value"); if(value==null) { value="&"+name+";"; } return "<input type='text' name='"+name+"' id='"+name+"' value='"+value+"'>";
