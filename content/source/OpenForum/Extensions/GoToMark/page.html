Parameters

  • pageName = The name of the page to go to (optional default = current page)
  • name = The name of the mark to scroll to
  • text = The text to display on the link

Usage:

[{GoToMark pageName="/OpenForum/Extensions/MarkPage" name="TestMark" text="Goto the test mark on the page MarkPage"}] Goto the test mark on the page MarkPage

renderer.sjstargetPageName = extension.getAttribute("pageName"); if(targetPageName ==null) { targetPageName = "/"+pageName; } name = extension.getAttribute("name"); text = extension.getAttribute("text"); return "<a href=\""+targetPageName+"#"+name+"\">"+text+"</a>";
